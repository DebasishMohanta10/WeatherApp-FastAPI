Asynchronous Restful Weather Service made using Fast API

Endpoints:

  • GET current weather in the city
    • GET /api/weather/city?country={country}&units={units}
  • Parameters

    • Required city={city} The name of the city to retrieve the weather

    Optional country={country} The name of the country where city is located

    Optional units={units} - (metric,imperial, standard), default is metric
